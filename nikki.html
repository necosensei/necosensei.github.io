  なだらかなカーブと横縞が見える気がする.







- ソース (pari/gp).

n = 1000 ;

D(m)    = return(if(m%4 == 1, m, 4*m)) ;
W(m)    = return(if(m == -1,4,if(m == -3,6,2))) ;
chi(d,a) = return(kronecker(d,a)) ;

H(m) = {
  my(d = abs(D(m)), w = W(m), a) ;
  return(-(w/(2*d))*sum(a=1,d-1,chi(-d,a)*a)) ;
}

{
  forstep(m = -1, -n, -1,
    if(issquarefree(m),
      h = H(m) ;
      print(abs(m)," ",h) ;
    );
  );
}

2017/06/03 (土)

- 今日は素数の統計的な偏りっぽいものが見える Ulam spiral.





- 縦横斜めの線は 2 次の素数生成多項式との事.



- ソース (pari/gp).

n = 10000 ;
{
  v = 0 ; x = real(v) ; y = imag(v) ;
  d = 1 ;

  maxx = minx = maxy = miny = 0 ;

  for(i=1,n,
    if(isprime(i), print(x," ",y)) ;
    v += d ; x = real(v) ; y = imag(v) ;
    if (x > maxx, maxx = x; d *= I) ;
    if (x < minx, minx = x; d *= I) ;
    if (y > maxy, maxy = y; d *= I) ;
    if (y < miny, miny = y; d *= I) ;
  );
}
